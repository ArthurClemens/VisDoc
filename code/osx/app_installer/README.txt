Instructions for updating the installer

Open XCode project
Update version in plist
Build XCode project

1. Unzip VisDoc_editable.dmg.zip
2. Open VisDoc_editable.dmg to mount the disk
3. Put a new version of VisDoc inside the folder
4. Close the second window, but leave the first window open
5. Open Disk Utility and in the left bar select the disk image VisDoc_editable.dmg
6. Choose from the main menu: Images > Convert... Enter the new version name, choose Image format: compressed and Encryption: none
7. Unmount the disk image
